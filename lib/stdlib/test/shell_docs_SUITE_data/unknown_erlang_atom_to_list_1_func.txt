[1m  atom_to_list(Atom)[0m

  Returns a list of unicode code points corresponding to the text
  representation of [4mAtom[0m.

  For example:

    > atom_to_list('Erlang').
    "Erlang"

    > atom_to_list('你好').
    [20320,22909]

  See [4municode[0m for how to convert the resulting list to different
  formats.
