[1m  list_to_tuple(List)[0m

  Returns a tuple corresponding to [4mList[0m, for example

    > list_to_tuple([share, ['Ericsson_B', 163]]).
    {share, ['Ericsson_B', 163]}

  [4mList[0m can contain any Erlang terms.
